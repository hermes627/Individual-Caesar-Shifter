# Individual-Caesar-Shifter
This is a bit of code to simulate a caesar shift where each letter is shifted by a different amount.

To use it, simply have your first input be the string of letters (uppercase or lowercase) you want to shift with no spaces, then input the integers representing the shifts for each letter, with spaces between each number.

If a letter is not shifted, use 0 as the input.
